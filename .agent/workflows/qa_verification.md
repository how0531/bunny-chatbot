# 🛡️ 完工驗證標準流程 (Mandatory QA Verification)

為了確保交付給長官的內容 100% 可執行且符合旗艦級品質，Kevin (QA) 必須執行以下驗證步驟：

1. **環境清理**：
   - 終結所有舊版背景程序（如 `taskkill /F /IM python.exe`），確保無衝突。
2. **部署檢查**：
   - 確認所有新修正的代碼已存檔。
   - 確認本地資源（如 `/static/libs`）已正確就緒。
3. **執行測試**：
   - 重新啟動 `python app.py`。
   - 使用 `curl` 或內部測試腳本驗證 API 回傳結構（JSON）是否正確。
4. **交互驗證**：
   - 在前端介面實測長官要求的具體指令（如「進 10 天市場強度」）。
   - 確認圖表渲染成功且無 JavaScript 錯誤。
5. **最終簽章**：
   - 由 Sophia 在 notify_user 中註明「品管驗證已完成」，並附上執行成功的依據。
